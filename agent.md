General Instructions

- This project is called VIBE-LA-YOUDRAWIT. It starts with a new github repo. 
- VIBE-LA-YOUDRAWIT is a js and Python dashboard project. You can use Typescript and React. The backend just needs an api.
- You have full control over the github repo structure, the node/python modules you install and how you minify for a distribution. You'll need to update the .gitignore file. If you ever need to use .env files make sure that these can't be committed for privacy reasons. Just don't overwrite the prompts folder.
- When you plan, make sure you do functional decomposition, use appropriate design patterns and write comprehensive tests for both frontend and backend. 
- Make sure the Readme.md file is up to date with how to install and use the lib 
- You are free to install libs that the project can use.
